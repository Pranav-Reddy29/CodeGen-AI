# Cohere AI Setup for CodeGen

## Why Cohere AI?

Cohere AI is an excellent choice for code generation because:
- **Fast & Reliable**: Optimized for code generation tasks
- **Cost-Effective**: Competitive pricing for API calls
- **High Quality**: Produces clean, production-ready code
- **Developer-Friendly**: Simple API with good documentation

## Setup Steps

### 1. Get Cohere API Key

1. Visit [Cohere AI](https://cohere.ai)
2. Sign up for a free account
3. Navigate to **API Keys** in your dashboard
4. Create a new API key
5. Copy the key (starts with `sk-...`)

### 2. Configure Environment

#### For Development:
```bash
# Copy the config template
cp server/config.env server/.env

# Edit server/.env and add your Cohere key:
COHERE_API_KEY=sk-your-actual-cohere-api-key-here
```

#### For Production:
```bash
# Copy the production template
cp env.production.template .env

# Edit .env and add your Cohere key:
COHERE_API_KEY=sk-your-actual-cohere-api-key-here
```

### 3. Install Dependencies

```bash
# Install all dependencies including Cohere SDK
npm run install:all
```

### 4. Test the Integration

1. Start your servers:
   ```bash
   npm run dev          # Frontend
   npm run server:dev   # Backend
   ```

2. Go to the Generator page
3. Enter a prompt like: "Create a React todo app with add, edit, and delete functionality"
4. Click "Generate Project"
5. You should see real code generated by Cohere!

## Cohere API Features Used

### Model: `command`
- Optimized for code generation
- Handles multiple programming languages
- Produces clean, well-structured code

### Configuration:
- **Max Tokens**: 3000 (sufficient for most projects)
- **Temperature**: 0.3 (consistent, reliable output)
- **Stop Sequences**: Prevents unwanted formatting
- **Response Cleaning**: Removes markdown formatting

## Troubleshooting

### Common Issues:

1. **"Failed to generate code with Cohere"**
   - Check your API key is correct
   - Verify you have sufficient credits
   - Check Cohere service status

2. **"Invalid API key"**
   - Ensure the key starts with `sk-`
   - Copy the key exactly as shown in Cohere dashboard
   - Check for extra spaces or characters

3. **"Rate limit exceeded"**
   - Cohere has rate limits on free tier
   - Consider upgrading your plan
   - Wait a few minutes and try again

4. **Poor code quality**
   - Try being more specific in your prompts
   - Specify the programming language clearly
   - Include requirements like "production-ready" or "with error handling"
